# Red Hat Connectivity Link — Reference Repository

Reference manifests and a minimal end-to-end example for teams implementing [Red Hat Connectivity Link (RHCL)](https://www.redhat.com/en/technologies/cloud-computing/openshift/connectivity-link) on OpenShift.

The goal of this repository is not to be deployed as-is in production, but to provide **copy-pasteable YAML examples** that you can adapt to your own cluster, DNS, namespaces, and application requirements.
